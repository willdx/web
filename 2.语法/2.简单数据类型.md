# 简单数据类型

## 简单数据类型

整数: 12345

浮点数: 3.1415926

字符串: “My name is Will!”

## 字符串连接

```python
a = "I am"
b = "will"
name = a + b

# 推荐使用join: 虽然语法复杂一些, 但习惯就好
# ''部分什么都不写表示直接连接, ','.join([a, b]) 表示用逗号连接a和b元素
name = ''.join([a, b])  
```


## 变量赋值
```python
a = '123'
b = a
print(b)
```
