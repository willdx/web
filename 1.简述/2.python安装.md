# Python安装


> Python是跨平台的, 这意味着它能运行在Windows, Mac以及各种Linux/Unix系统中;  

## Python2 or Python3

Python3并不能完全反向兼容Python2, Python3在Python2的基础上做了很多优化, 并且提供了很多新特性(PS: 具体后面再讲), 所以推荐使用Python3;


## MAC 安装Python3

```shell
brew install python3
pip3 install ipython
```

## Linux 安装Python3

```shell
pip3 install python3
pip3 install ipython
```
